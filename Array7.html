Que.7 Two elements whose sum is closest to zero


Explanation

        An Array of integers is given, both +ve and -ve.
        You need to find the two elements such that their
        sum is closest to zero.
        For the below array, program should print -80 and 85.

Approach : 


        For each element, find the sum of it with 
        every other element in the array and compare sums.
        Finally, return the minimum sum.

Code:



// C++ code to find Two elements
// whose sum is closest to zero
# include <bits/stdc++.h>
# include <stdlib.h> /* for abs() */
# include <math.h>
 
using namespace std;
void minAbsSumPair(int arr[], int arr_size)
{
    int inv_count = 0;
    int l, r, min_sum, sum, min_l, min_r;
     
    /* Array should have at least
       two elements*/
    if(arr_size < 2)
    {
        cout << "Invalid Input";
        return;
    }
     
    /* Initialization of values */
    min_l = 0;
    min_r = 1;
    min_sum = arr[0] + arr[1];
     
    for(l = 0; l < arr_size - 1; l++)
    {
        for(r = l + 1; r < arr_size; r++)
        {
        sum = arr[l] + arr[r];
        if(abs(min_sum) > abs(sum))
        {
            min_sum = sum;
            min_l = l;
            min_r = r;
        }
        }
    }
     
    cout << "The two elements whose sum is minimum are "
         << arr[min_l] << " and " << arr[min_r];
}
 
// Driver Code
int main()
{
    int arr[] = {1, 60, -10, 70, -80, 85};
    minAbsSumPair(arr, 6);
    return 0;
}

Output: 



The two elements whose sum is minimum are -80 and 85


Time Complexity


O(n^2) where N is the number of elements present in the array. Here we run one loop .


Space Complexity


O(1) because we don't use any auxiliary space we just use start and end variables to swap the array.
