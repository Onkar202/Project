Que.3 Recursive Bubble Sort


Explanation

        Example: 
        First Pass: 
        ( 5 1 4 2 8 ) –> ( 1 5 4 2 8 ), Here, algorithm compares the first two 
        elements, and swaps since 5 > 1. 
        ( 1 5 4 2 8 ) –> ( 1 4 5 2 8 ), Swap since 5 > 4 
        ( 1 4 5 2 8 ) –> ( 1 4 2 5 8 ), Swap since 5 > 2 
        ( 1 4 2 5 8 ) –> ( 1 4 2 5 8 ), Now, since these elements are already 
        in order (8 > 5), algorithm does not swap them.

Code:



// C/C++ program for recursive implementation
// of Bubble sort
#include <bits/stdc++.h>
using namespace std;
 
// A function to implement bubble sort
void bubbleSort(int arr[], int n)
{
    // Base case
    if (n == 1)
        return;
 
    // One pass of bubble sort. After
    // this pass, the largest element
    // is moved (or bubbled) to end.
    for (int i=0; i<n-1; i++)
        if (arr[i] > arr[i+1])
            swap(arr[i], arr[i+1]);
 
    // Largest element is fixed,
    // recur for remaining array
    bubbleSort(arr, n-1);
}
 
/* Function to print an array */
void printArray(int arr[], int n)
{
    for (int i=0; i < n; i++)
        printf("%d ", arr[i]);
    printf("\n");
}
 
// Driver program to test above functions
int main()
{
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int n = sizeof(arr)/sizeof(arr[0]);
    bubbleSort(arr, n);
    printf("Sorted array : \n");
    printArray(arr, n);
    return 0;
}


Output:



Sorted array : 11 12 22 25 34 64 90



Time Complexity


O(n*n)


Space Complexity


O(n)
